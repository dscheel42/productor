## Getting Started 
``` 
## Go to the working directory for productor
cd productor

Rscript update_env.R # ONLY AFTER UPDATING .productor.conf
Rscript scripts/build.R # Take a nap, runs for a while
Rscript scripts/start.R
```

## Stopping
```
Rscript scripts/stop.R
```
